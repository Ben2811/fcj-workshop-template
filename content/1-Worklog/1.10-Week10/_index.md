### Week 10 Objectives:

* Learn container hosting options with Amazon EKS.
* Learn to automate CI/CD pipelines with AWS CodePipeline and GitHub integration.
* Practice migrating a monolithic application to a microservices architecture using Docker and Fargate.
* Build a serverless frontend integrated with Amazon API Gateway.
* Get started with deploying serverless applications using the AWS Serverless Application Model (SAM).
