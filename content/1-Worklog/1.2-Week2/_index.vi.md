### Mục tiêu tuần 2:

* Thực hành với Amazon EC2: instance metadata, user data và các khái niệm nền tảng của Auto Scaling.
* Thực hành khởi chạy, chỉnh sửa và backup EC2 instance (snapshot & custom AMI).
* Triển khai một ứng dụng thực tế trên Amazon EC2 và quản lý quyền truy cập bằng IAM user, access key, role.
* Làm quen với môi trường phát triển trên cloud với AWS Cloud9, và học cách khởi tạo database với Amazon RDS.

### Kết quả đạt được tuần 2:

* Hoàn thành bài lab thực hành với Amazon EC2, bao gồm instance metadata, user data và Auto Scaling.
* Thực hành khởi chạy, chỉnh sửa và backup EC2 instance bằng snapshot và custom AMI.
* Triển khai thành công AWS User Management Application trên Amazon Linux.
* Tạo và sử dụng IAM access key, thực hành tạo và sử dụng IAM Role.
* Khám phá AWS Cloud9 - IDE chạy ngay trên trình duyệt.
* Tạo database trên Amazon RDS.
* Tự động hoá việc scale hạ tầng với Amazon EC2 Auto Scaling.
* Xây dựng dashboard giám sát hệ thống bằng Amazon CloudWatch.
* Thực hành sử dụng AWS CLI để quản lý Amazon EC2 instance trên cả Windows và Ubuntu.

**Dịch vụ AWS đã học trong tuần:** Amazon S3, Amazon EC2, Amazon EC2 Auto Scaling, Amazon RDS, AWS CLI, Amazon Route 53, Amazon CloudWatch, AWS Cloud9, IAM (Users, Access Keys, Roles).

**Kỹ năng đạt được:** Sử dụng AWS CLI; cấu hình, backup và đóng gói virtual server.
