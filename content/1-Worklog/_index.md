I completed the **Workforce Bootcamp – First Cloud AI Journey (FCJ)** internship program at **Amazon Web Services Vietnam Co., Ltd.**, over **12 weeks** from 17/04/2026 to 10/07/2026. The first 9 weeks followed the AWS Study Group curriculum to build up core AWS skills; the last 3 weeks (Week 10–12) were spent applying that knowledge hands-on to a real team project, **PeriodIQ**, and writing up the resulting AWS Workshop chapter of this report.
