### Mục tiêu tuần 4:

* Mở rộng kiến thức AWS networking: VPC Peering và Transit Gateway.
* Tìm hiểu kiểm soát truy cập theo tag (tag-based access control) và tổ chức tài nguyên với IAM Tags và Resource Groups.
* Xây dựng dashboard giám sát nâng cao bằng cách tích hợp Amazon CloudWatch với Grafana.
* Làm quen với quản lý instance từ xa an toàn bằng AWS Systems Manager (Session Manager).

### Kết quả đạt được tuần 4:

* Kết nối nhiều VPC lại với nhau bằng VPC Peering.
* Quản lý tập trung các kết nối mạng bằng AWS Transit Gateway.
* Quản lý quyền truy cập EC2 và tổ chức tài nguyên bằng IAM Tags và Resource Groups.
* Xây dựng dashboard giám sát nâng cao bằng cách tích hợp Amazon CloudWatch với Grafana.
* Làm quen với AWS Systems Manager (Session Manager) để truy cập instance an toàn mà không cần SSH key.

**Dịch vụ AWS đã học trong tuần:** Amazon VPC Peering, AWS Transit Gateway, AWS IAM (Tag-based access control), AWS Resource Groups, Amazon CloudWatch & Grafana, AWS Systems Manager (Session Manager).

**Kỹ năng đạt được:** Thiết lập networking nâng cao và định tuyến traffic; quản lý quyền truy cập bảo mật chi tiết bằng attribute-based access control (ABAC); giám sát hạ tầng nâng cao và trực quan hoá dashboard; quản lý truy cập từ xa (Session Manager).
