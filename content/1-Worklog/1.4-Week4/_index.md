### Week 4 Achievements:

* Linked multiple VPCs together using VPC Peering.
* Centralized network connections using AWS Transit Gateway.
* Managed EC2 service access and organized resources using IAM Tags and Resource Groups.
* Built an advanced monitoring dashboard by integrating Amazon CloudWatch with Grafana.
* Got introduced to AWS Systems Manager (Session Manager) for secure, SSH-key-free instance access.

**AWS services learned this week:** Amazon VPC Peering, AWS Transit Gateway, AWS IAM (Tag-based access control), AWS Resource Groups, Amazon CloudWatch & Grafana, AWS Systems Manager (Session Manager).

**Skills gained:** Advanced cloud networking setup and traffic routing; fine-grained security access management using attribute-based access control (ABAC); advanced infrastructure monitoring and dashboard visualization; remote access management (Session Manager).
